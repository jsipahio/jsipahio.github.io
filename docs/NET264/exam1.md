# NET264 - Unix/Linux System Administration: Exam 1 Study Guide
## General
Review the quizzes. The questions on the exam will be similar.
## Week 1
- Understand the relationship and differences between the kernel and distributions
## Weeks 2-3
- Explain the input and output streams, and how to access them
- Identify methods for redirecting terminal I/O
- Identify shell globbing characters
- Identify the usage of common commands
- Explain how to create variables in bash
- Explain when/why you'd use Python/Ruby over bash
- Describe the standard casing (i.e., lowercase vs uppercase, how to separate words) for local and environment variables
- Explain the basics of writing a shell script
- Identify the 6 principles you are recommended to follow for scripting
- Describe the basic constructs (while loops, for loops, functions, if statements, etc.) of scripts
- Explain how to run a script
- Identify why you would use REGEX expressions
## Week 4
- Explain the basics of booting and shutting down UNIX-like operating systems
- Describe GRUB (what it stands for and what it does)
- UEFI vs. BIOS
- Describe `systemd` and how it differs from `init`
- Identify access control mechanisms
- Describe `sudo`, including how to add/remove users who can utilize sudo
- Describe the most important system targets (used to be run levels) for system admins
- Describe the commands to shut down a UNIX-like system
## Week 5
- Identify the different states of processes
- Identify common types of processes
- Explain what causes kernel panics
- Identify which users may send signals to processes
- Describe the responsibilities of the OS kernel
- Identify tools for monitoring processes
## Weeks 6-7
- Identify common package managers
