# DSA Chapter 10: Heap, Priority Queue, and Heap Sort

This chapter focuses on the Heap data structure, which is a balanced binary tree. Heaps are often used to implement priority queues, a special type of queue that moves elements with the highest priority to the front of the queue. Finally, we discuss how priority queues and heap can be used to implement a sorting algorithm called "heap sort".
