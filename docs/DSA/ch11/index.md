# DSA Chapter 11: Naive Sorting Algorithms

The last chapter introduced heap sort, which is a time efficient sorting algorithm (however, its speed is achieved at the cost of needing a large amount of extra space). This chapter takes a step back and looks at the conceptually simpler sorting algorithms: insertion, bubble, and selection. While simpler to understand and program, these sorting algorithms are slower than heap sort and the more advanced algorithms that will be introduced next week.
