# Graphs and Graph Representations

A graph is a data structure that represents interconnected nodes. Trees, which we saw previously, are graphs that are constrained to contain a root element, and all nodes must have only one parent. Trees are a type of directed graphs. The graphs we discuss in this chapter are undirected graphs. We call the nodes of the graph vertices and the links between nodes edges. In an undirected graph, an edge between two vertices can be traversed in any direction.
