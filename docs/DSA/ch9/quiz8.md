# Quiz 8
Submit a PDF, Word Document, or Text Entry with the answers to these questions.

## Questions
1. True/False: Recursive functions will never cause a stack overflow. (1 point)

2. Short Answer: Explain how a binary tree differs from a linked list. (3 points)

3. True/False: A tree can have more than two children. (1 point)

4. Multiple Choice: A binary tree can be implemented using: 
- A. Set
- B. Hash Table
- C. Array
- D. Dictionary

5. True/False: A binary search tree is guaranteed to provide O(log(n)) search times.

6. Multiple Choice: Which is not a self-balanced binary tree:
- A. Complete Binary Tree
- B. Red-Black Tree
- C. AVL Tree
- D. Binary Heap

7. Short Answer: List the properties of a complete binary tree. (3 points)

8. Multiple Choice: Which is not a property of a red-black tree. (1 point)

9. Short Answer: I want to insert numbers from 1 to 100 into a binary tree, keeping the tree as balanced as possible. What order should I insert the elements? (3 points)
