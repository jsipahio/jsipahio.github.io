# Recursion

Recursion is when a function calls itself. Defining recursive functions will be useful for tree and graph traversals, which we will be discussing in the coming chapters. Therefore, it is important to introduce recursion now. One way to repeatedly execute a set of instructions, and the technique we have been using through the course so far, is to write either a while or a for loop. However, in some cases it can be easier and/or more efficiently using recursion.
