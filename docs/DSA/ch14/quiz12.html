Questions

  1. True/False: A digraph cannot have any cycles. (1 point)

  2. Multiple Choice: Which algorithm is used to find the shortest path in a graph: (1 point) A. Dijkstra B. Kruskal C. Ford-Fulkerson D. Kahn

  3. Multiple Choice: Which algorithm is used to find the max flow in a graph: (1 point) A. Dijkstra B. Kruskal C. Ford-Fulkerson D. Kahn

  4. Multiple Choice: Which algorithm is used to perform a topological sort in a graph: (1 point) A. Dijkstra B. Kruskal C. Ford-Fulkerson D. Kahn

  5. True/False: Greedy algorithms split a problem into manageable subproblems. (1 point)

  6. Short Answer: Assume we have the following jobs, their pay, and start/end times. Use a greedy algorithm to pick the most sequence of jobs. (5 points)

  7. Short Answer: The fractional knapsack problem is as follows: given a unordered list of (item, weight, and value) triplets, and a maximum weight W, what is the optimal quantity of items to take. Assume you have the following struct:

struct Item {
    String name;
    double weight;
    double value;
}

Assume you have an array<Item> items that contains unordered items. Write a pseudocode algorithm that finds the maximum value you can take, while staying under weight W. In the fractional knapsack problem, you are allowed to take a partial item. What is the time complexity of this algorithm? (5 points)
