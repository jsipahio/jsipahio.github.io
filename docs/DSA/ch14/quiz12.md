```cpp
struct Item {
    String name;
    double weight;
    double value;
}
```
Assume you have an `array<Item> items` that contains unordered items. Write a pseudocode algorithm that finds the maximum value you can take, while staying under weight W. In the fractional knapsack problem, you are allowed to take a partial item. What is the time complexity of this algorithm? (5 points)
