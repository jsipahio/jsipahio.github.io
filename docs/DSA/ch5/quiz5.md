# Quiz 5
Submit a PDF, Word Document, or Text Entry with the answers to these questions.

## Questions

1. True/False: Generic types **must** be defined in a separate C++ file.

2. Short Answer: Provide the declaration for a generic function add that can potentially have different types for its arguments and return (i.e., the function can be called with an int and float as arguments and return a double). (3 points)

3. Multiple Choice: Which of the following is not a keyword used for declaring a template parameter? (1 point)
- A. `class`
- B. `classname`
- C. `typename`

4. True/False: Functions, classes, and global variables can have generic parameter lists applied to them. (1 point)

5. Short Answer: Write the template parameter list for this class: (3 points)
```cpp
Array<int, 5, 20> array;
```

6. Multiple Choice: In C++, inheritance by classes _____ by default? (1 point)
- A. `protected`
- B. `private`
- C. `public`
- D. `package`

7. Short Answer: Write a base class Mammal with two derived classes, Cat and Dog. Make up data for the animal class, but make sure it is visible to Cat and Dog, but not accessible to outside entities. (5 points)
