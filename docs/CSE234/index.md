# CSE234 Home
Additional information for the CSE234: Advanced C++ course at Stark State College. This provides an overview of the C++ Standard Template Library (STL). To view the full details of the STL, check https://cppreference.com/ and https://cplusplus.com/. CPP Reference has more in-depth information on the history and changes of the STL, cplusplus has more examples. In general, information here is assuming C++11 or newer. If a feature that is discussed requires a newer version of C++, it will be marked with the version in parenthesis after the statement. For example: `std::any` can be used as the template type to a `std::vector` to create a Python-like list of mixed types (C++17).
